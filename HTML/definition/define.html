HTML Basics

HTML

HTML stands for HyperText Markup Language. It is used to create and design webpages. It structures the content on the web using elements and tags.

Tags

Tags are special keywords written inside angle brackets like <html> and <p>. They tell the browser how to display content.

Elements

An element consists of a start tag, content, and an end tag. Example: <p>This is a paragraph</p> is a paragraph element.

Semantic Elements

Semantic elements clearly describe their meaning to both the browser and developer. Examples include <header>, <footer>, <article>, and <section>.

Non-Semantic Elements

Non-semantic elements do not give information about their content. Examples include <div> and <span>.

Block Level Elements

Block-level elements take full width and start on a new line. Examples include <div>, <p>, <h1> to <h6>.

Inline Elements

Inline elements only take as much width as needed and do not start on a new line. Examples include <span>, <a>, <strong>, <em>.

Attributes

Attributes provide extra information about HTML elements. They are always written inside the opening tag. Example: <a href="https://example.com">Link</a>

Forms

HTML forms are used to collect user input like name, email, and password. They include elements like input, textarea, button, and select.

HTML5

HTML5 is the latest version of HTML. It supports new features like video, audio, semantic elements, and improved form controls.

Audio Element

The audio element is used to embed sound content like music or recordings in a webpage.

Video Element

The video element is used to embed video content in a webpage.

Iframe Element

The iframe element is used to display another webpage inside the current webpage.
